Especialidade

HTML

HTML é a sigla para HyperText Markup Language, que significa Linguagem de Marcação de Hipertexto. É a linguagem padrão para criar páginas da web e é o componente base da internet.

CSS

CSS é a sigla para Cascading Style Sheets, que significa Folha de Estilo em Cascata. É uma linguagem de programação que define a aparência de páginas da web. O CSS é usado em conjunto com o HTML.

JAVASCRIPT

JavaScript (JS) é uma linguagem de programação que permite adicionar interatividade a páginas da web. É uma das linguagens mais populares do mundo e é usada para criar jogos, aplicativos e outros conteúdos.
